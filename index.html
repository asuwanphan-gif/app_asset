หน้าภาพรวม (Dashboard)
  • แสดงสถิติครุภัณฑ์ทั้งหมด แยกตามสถานะ (ใช้งาน / ชำรุด / ไม่พบ / เกินอายุ)
  • กรองข้อมูลได้ตาม ปีงบประมาณ → สาขา → สถานะ
  • แสดง banner รอบตรวจที่กำลังดำเนินการอยู่ พร้อมเปอร์เซ็นต์ความคืบหน้า
รายการครุภัณฑ์
  • ค้นหาด้วยชื่อ รหัส หรือ Serial No. · กรองตามปีงบ สาขา สถานะ และ รหัสครุภัณฑ์
  • เพิ่มครุภัณฑ์ใหม่พร้อมรูปถ่ายตรวจรับ 2 รูป
  • คลิกรายการเพื่อดูรายละเอียด (พ.ด. 2) แก้ไข เพิ่มประวัติซ่อม/โอนย้าย
  • นำเข้าข้อมูล — อัปโหลด Excel ตาม template เพื่อนำเข้าหลายรายการพร้อมกัน
  • นำเข้ารูปภาพ — อัปโหลดรูปหลายไฟล์พร้อมกัน ระบบ match ชื่อไฟล์กับรหัสครุภัณฑ์อัตโนมัติ
    → ใช้ร่วมกับ extract_photos.py เพื่อดึงรูปจากไฟล์ Excel พ.ด. 2
การตรวจนับครุภัณฑ์ประจำปี
  • Admin: สร้างรอบตรวจ กำหนดปีงบประมาณและช่วงวันที่ · แจ้งสาขาทางอีเมล (ต้องกรอก email สาขาก่อน) · กำกับติดตามความคืบหน้ารายสาขา · ปิดรอบตรวจเมื่อเสร็จสิ้น
  • สาขา: กด "กรอกผล" เพื่อบันทึกผลการตรวจรายการ · เลือกสถานะแต่ละรายการแล้วกด "บันทึกทั้งหมด"
  • ดูประวัติรอบตรวจทั้งหมดได้ที่ปุ่ม "ประวัติรอบตรวจ"
ซ่อมบำรุง · โอนย้าย · จำหน่าย
  • ซ่อมบำรุง: บันทึกวันที่ซ่อม ค่าใช้จ่าย ผู้ซ่อม ดูประวัติในแต่ละรายการ (Admin ลบประวัติได้)
  • โอนย้าย: โอนครุภัณฑ์ข้ามสาขา ระบบบันทึกประวัติทุกครั้ง (Admin ลบประวัติได้)
  • จำหน่าย: เปลี่ยนสถานะเป็น "รออนุมัติจำหน่าย" → Admin อนุมัติ → สถานะเปลี่ยนเป็น "จำหน่ายแล้ว"

ขั้นตอน: Extract รูปจาก Excel
  1. ดาวน์โหลดไฟล์ extract_photos.py ใส่ไว้ในโฟลเดอร์เดียวกับไฟล์ Excel
  2. เปิด Command Prompt แล้วรัน: python extract_photos.py
  3. ได้โฟลเดอร์ extracted_photos/ — รูปชื่อตามรหัสครุภัณฑ์
  4. กลับมาหน้านี้ → เลือกรูปทั้งหมดจากโฟลเดอร์นั้น → อัปโหลด
